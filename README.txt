Ce script python permet de crypter et décrypter une chaîne de caractères par l'intermédiaire du code César.

Cela consiste à remplacer chacune des lettres par une autre en fonction du "décalage" préalablement choisi.
Par Exemple, lorsque la lettre "a" subit un décalage de 3, elle devient alors la lettre "d".
Il faut par ailleurs reprendre au début de l'alphabet lorsque le décalage appliqué à une lettre est plus important que la 
longueur de l'alphabet.

Ce script permet aussi de décrypter un message codé à l'aide du code césar en soustrayant cette fois le décalage à la 
lettre. Cependant il est nécessaire de posséder la valeur du décalage (le test par la force-brute étant trop gourmand en 
ressources).

J'ai effectué ce script car j'apprécie étudier les différentes méthodes d'encryptage, notamment celles qui ont traversé 
l'histoire.
